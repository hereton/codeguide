#Code guidance.

## 1) DRY (Don't repeat yourself)
- It's means don't write duplicate code. It's important not to abuse it, duplication is not for code, but for functionality .
for example 

``` if(1>2){
		return true; --> duplicate code.
		}
		else{
		return false; --> dupliecate code.
		}
		
		```
